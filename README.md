# webpack-simple-react-typescript

Webpack 4 simple boilerplate for React 16 with Typescript 3, Jest & SASS

![Webpack simple](webpack-simple.jpg)

# Includes
- React 16
- React router 4
- Typescript 3 loader
- Sass loader
- Postcss loader
- Bundling into HTML plugin
- Dev server with hot reloading

# Instructions
- Clone this repo
- `npm install`
- `npm start`
- `npm run test:watch`
